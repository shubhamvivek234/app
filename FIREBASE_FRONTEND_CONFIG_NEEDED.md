Please provide the Firebase Config object for the Frontend.
		It should look like this:
		```javascript
		const firebaseConfig = {
		  apiKey: "...",
		  authDomain: "...",
		  projectId: "...",
		  storageBucket: "...",
		  messagingSenderId: "...",
		  appId: "..."
		};
		```
		You can find this in your Firebase Console under Project Settings -> General -> Your apps -> Web app.
